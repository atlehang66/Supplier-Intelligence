# Sourcing Intelligence

## Overview

Sourcing Intelligence is a business intelligence project designed to evaluate, compare, and rank international suppliers using data-driven methods.

The project supports sourcing decisions for an e-commerce business by combining supplier information from multiple sources with analytics, scoring models, and interactive dashboards.

The objective is to reduce sourcing risk, improve supplier selection, and provide data-backed insights for procurement and investment decisions.

---

## Objectives

- Build a centralized supplier database.
- Evaluate supplier reliability using measurable criteria.
- Analyze pricing, shipping, and lead times.
- Calculate estimated landed costs and profit margins.
- Develop supplier scoring models.
- Produce interactive Power BI dashboards.
- Support sourcing decisions with business intelligence.
